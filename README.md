# Fatmug

Fatmug is a web application made in python/Django. It is a part of developer assignment (Vendor Management System). Document [link](https://drive.google.com/file/d/1hLWc35hpLQ0EPDAZjZloomeAxgn3V7g9/view)

## Installation

Clone the repository
```bash
git clone https://github.com/ChetanSureka/Fatmug.git
```
Install the required libraries from requirements.txt
```bash
cd fatmug
pip install -r requirements.txt
```
Run the django app.
```bash
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```
